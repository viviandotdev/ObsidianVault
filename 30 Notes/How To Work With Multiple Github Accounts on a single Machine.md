```jsx
ssh-keygen -t rsa -C "[linvivian61@gmail.com](<mailto:linvivian61@gmail.com>)" -f "github-vivian-personal"
ssh-keygen -t rsa -C "[linvivian61@gmail.com](<mailto:linvivian61@gmail.com>)" -f "github-vivian-personal"
```

```jsx
ssh-add -K ~/.ssh/github-vivian-personal
```

```jsx
clone the repo
```
